## Getting Started

To setup a local version of the repo please follow the following

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/hawyar/monster-link.git
```

2. Install NPM packages

```sh
npm install
# yarn
```
